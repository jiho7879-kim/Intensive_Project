▸ WHY UNEXPLAINED IS HIGH
  • 챔버 컨디션 (Chamber Condition) — 동일 파라미터라도 챔버 내벽 오염도, 플라즈마 균일성이 배치마다 다름
  • 포토마스크 상태 (Mask Degradation) — 마스크 노후화에 따른 선폭 변화는 로트 단위로 누적
  • 웨이퍼 결정 방향 편차 — 동일 로트 내에서도 웨이퍼 간 결정 방향 오차가 존재
  • 온도 이력 (Thermal History) — 전 공정 누적 열 이력은 현재 데이터에 미포함
  • 오염 입자 분포 — 결함 수(defect_count)는 포착되지만 크기·위치 분포는 미측정
  • 설비 간 마이크로 차이 — 동일 모델 설비라도 개체 간 미세 편차 존재
▸ ACTIONABLE RECOMMENDATIONS
  • 22nm 최우선: 현재 n=125 → 최소 500개 이상 수집 시 R² 0.4+ 기대
  • 챔버 상태 지표 추가: 챔버 클리닝 이후 누적 RF 시간, 내벽 온도 편차
  • 마스크 이력 피처: 마스크 사용 횟수, 마지막 검사 이후 경과 로트 수
  • 웨이퍼 ID 연계: 동일 잉곳(Ingot) 내 위치 정보로 결정 방향 편차 추정
  • 공정 이력 집계: 전 N개 공정 step의 이탈 횟수 누적 피처 생성
  • 인라인 계측 확대: 현재 최종 수율만 있는 구조 → 중간 검사 데이터 연계
